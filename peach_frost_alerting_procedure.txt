PEACH FROST ALERTING BACKUP PROCEDURE
====================================

Goal
----
Back up the Ascend flow:

flows/peach_frost_alerting/

and its local dataset:

data/read_farmers.csv

into the local repo folder:

C:\Users\ISBE\ascend-pipelines


PART 1 — CREATE LOCAL FOLDERS
-----------------------------
Open Command Prompt and run:

mkdir C:\Users\ISBE\ascend-pipelines\flows\peach_frost_alerting
mkdir C:\Users\ISBE\ascend-pipelines\flows\peach_frost_alerting\components
mkdir C:\Users\ISBE\ascend-pipelines\data

If folders already exist, that is okay.


PART 2 — FILES TO COPY
----------------------

-------------
1.

2.
Ascend path:
flows/peach_frost_alerting/README.md

Save as:

PART 3 — HOW TO COPY EACH FILE
------------------------------
For each file in Ascend:

1. Open the file in the Ascend editor
2. Click inside the editor
3. Press Ctrl + A
4. Press Ctrl + C

Then on your computer:

1. Open Notepad or VS Code
2. Press Ctrl + V
3. Save the file in the correct local path
4. Use the exact same file name


PART 4 — RECOMMENDED COPY ORDER
-------------------------------
Copy files in this order:

1. flows/peach_frost_alerting/peach_frost_alerting.yaml
2. flows/peach_frost_alerting/README.md
3. flows/peach_frost_alerting/components/read_farmers.yaml
4. data/read_farmers.csv
5. flows/peach_frost_alerting/components/build_frost_risk_model.py

PART 5 — VERIFY LOCAL FILES
---------------------------
After copying, run:

dir /s C:\Users\ISBE\ascend-pipelines\flows\peach_frost_alerting
dir /s C:\Users\ISBE\ascend-pipelines\data

This lets you confirm the files exist.
